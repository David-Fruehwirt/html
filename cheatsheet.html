Cheat sheet

Syntax:

  • HTML (HyperText Markup language): Language used to create web pages
  • URL (Uniform Recource Locator): Unique adress for web resources
  • HTTP (HyperText Transfer Protocol): Protocol for data transmission over the web

What does "DOCTYPE" do?

Tells the website what document type to expect, for example:

<!DOCTYPE html > tells the website that it uses the type html.

What is a "body element"?

Defines the body of the html document, so it has the content such as headers, paragraphs, etc.

What does the "em element" do?

Converts text into emphasized text which is mostly displayed in italic.

Example: I like programming!

What is a "head element"?

A head contains metadata (data about data) for example: <style> used to change fonts or colors.

What does the "html element" do?

Contains all html contents (elements) except DOCTYPE.

What does a "strong element" do?

Converts text into a strongly marked text for example:

I like programming!

What does the "title element" do?

Used in the head element, defines a title in browser toolbar and displays a title for the page in search-engine results.
